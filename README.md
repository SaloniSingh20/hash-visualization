# 🚀 Hashing Simulator  

An interactive web-based tool to visualize how **hash tables** work, including insertion, deletion, searching, and collision handling techniques.

---

## 🌐 Live Demo  
https://hashing-simulator-theta.vercel.app  

---

## 📌 Features  

- Visualizes hash table operations step-by-step  
- Supports:
  - Chaining  
  - Linear Probing  
  - Double Hashing  
- User-friendly interface  
- Real-time updates and interaction  

---

## 🧠 Concepts Demonstrated  

- Hash Functions  
- Collision Resolution  
- Open Addressing  
- Closed Addressing  
- Load Factor  

---

## 🛠️ Tech Stack

## ⚙️ How to Run  

### Clone the repo  
```bash
git clone https://github.com/SaloniSingh20/hashing-simulator.git
cd hashing-simulator
```

### Run frontend  
Open `web/index.html`  

### Run C++ (optional)  
```bash
g++ main.cpp -o hash_demo
./hash_demo
```
